# 快速上手

## 引入

```javascript
import Vue from 'vue'
import BaiduMap from 'vue-baidu-map'

Vue.use(BaiduMap, {
  // ak 是在百度地图开发者平台申请的密钥 详见 http://lbsyun.baidu.com/apiconsole/key */
  ak: 'YOUR_APP_KEY'
})
```

## 挂载

```html
<template>
  <baidu-map class="map-view">
  </baidu-map>
</template>

<style>
.map-view {
  width: 400px;
  height: 300px;
}
</style>
```
